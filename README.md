# 📘 Personal Journal Manager
---

## ✨ Overview

**Personal Journal Manager** is a simple Python project that allows users to write, read, search, and delete journal entries using file handling.
It is designed for beginners to understand core Python concepts easily.

---

## 🚀 Features

✔️ Add new journal entries
✔️ View all saved entries
✔️ Search entries using keywords
✔️ Delete all entries
✔️ Error handling using `try-except`

---

## 🛠️ Tech Stack

* 🐍 Python
* 📂 File Handling
* ⚠️ Exception Handling
